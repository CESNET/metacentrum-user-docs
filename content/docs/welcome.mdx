Welcome to the MetaCentrum grid computing service documentation.

- To access this service, please refer to the [How to get an account](./access/account) section.
- To see an overview of the grid's infrastructure, go to the [Basic concepts](./computing/concepts) page.
- Basic guidelines for running jobs are described in [Userguide on running jobs](./computing/run-basic-job).
- For information on the rules for working with data, see the [How to work with data](./data/types-of-storage) chapter.
- A list of the most commonly used software available in MetaCentrum can be found in the [Software](./software/alphabet) section.
- Contact [user support](./support/) with any enquiries.
- If you notice any errors in this documentation or if anything is missing, please open an issue or help us fix it in our [GitHub repository](https://github.com/CESNET/metacentrum-user-docs).
