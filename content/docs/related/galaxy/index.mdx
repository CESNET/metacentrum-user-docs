[Galaxy](https://galaxyproject.org/) is a web-based platform designed for running computational and statistical analyses with focus on openness and usage of FAIR data. It originally started in biomedical science but nowadays spans numerous scientific domains including ecology, natural language processing, chemistry, climate science, and social sciences.

<Callout type="info" title="Czech Galaxy Server -- usegalaxy.cz" icon="ðŸ’¡">
Metacentrum maintains a full-fledged [Galaxy server](../graphical/usegalaxy) open to all members of the Czech academia backed by the national computational infrastructure.
</Callout>

In addition to the [usegalaxy.cz](https://usegalaxy.cz) server we also manage the specialized [RepeatExplorer](../related/galaxy/repeat_explorer) Galaxy instance which is open worldwide thanks to the funding from ELIXIR Europe. And last, for the group at [RECETOX](https://www.recetox.muni.cz/en), we run the Untargeted Mass Spetrometry Analysis Galaxy server -- [UMSA](../related/galaxy/umsa).

If you need any help or experience tool errors or have unexpected issues with any of the Galaxy instance above please contact us at galaxy@cesnet.cz.
