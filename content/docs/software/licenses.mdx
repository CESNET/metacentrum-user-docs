title: Licensed software
---

Some software installed in MetaCentrum has such a license that each user must agree with the license. Such software is installed in a system directory that is accessible only by users who are member of the appropriate groups starting with `lic_`.

You can express your agreement with the license by asking for being added to the respective groups at the links below. Clicking on any of the software listed will redirect you to Perun interface.
